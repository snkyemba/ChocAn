Team Analysis Paragraph

The sections that will be computerized are all of the controllers- the roster edit controller, the report generation controller, the service log controller, and the directory controller. These will all be computerized through programs that take in the user’s input and produce an output, usually a report. The data of the users will be stored in various reports that can be accessed through different controllers. The hardware needed is various user PC’s, along with a server to store the reports, and also various card readers in order to check provider’s credentials. The ChocAn system will interact with other systems through the compilation of reports and with the timer to compile those reports on a weekly basis. The users will interact with the system through inputs on their personal hardware.
